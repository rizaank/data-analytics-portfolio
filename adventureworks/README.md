# AdventureWorks SQL Practice

T-SQL practice queries written against the AdventureWorks2022 
database running in SQL Server via Docker on local machine.

## What This Covers
- Multi-table joins across 3-5 tables
- Aggregations with GROUP BY and HAVING
- CASE WHEN with NULL handling and ELSE clauses
- Window functions: RANK, DENSE_RANK, ROW_NUMBER, LAG, LEAD, 
  SUM OVER, AVG OVER, MAX OVER
- Single and chained CTEs
- Deduplication patterns
- Benchmark questions completed without hints under 
  interview-style conditions

## Files
| File | Topics Covered |
|------|---------------|
| 01_joins_and_aggregations.sql | JOINs, GROUP BY, HAVING |
| 02_case_when.sql | CASE WHEN, ELSE, NULL handling |
| 03_window_functions.sql | All window functions |
| 04_cte_practice.sql | CTEs, chaining, deduplication |
| 05_benchmark_questions.sql | Mixed problems, no hints |

## Environment
- SQL Server 2022 via Docker
- VS Code with SQL Server extension
- macOS (Apple M4)
